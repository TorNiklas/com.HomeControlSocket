# Develco Home Control Outlet
This is for the double outlet, also known as HCO, HCO 2, HCOHCD.

## Flow cards 

### Device: Develco Home Control Outlet

#### Triggers
- Outlet is turned on
- Outlet is turned off
- Temperature changed

#### Conditions
- Outlet is turned on
- Outlet is turned off
- Temperature is above value

#### Actions
- Turn outlet on
- Turn outlet off
- Toggle outlet
